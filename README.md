# 概要
タスクの追加、削除、完了ができる最低限の機能を搭載したTodoアプリ。

# 作成目的
Reactを用いたアプリ開発は初めてのため、難易度が低いアプリのほうが良いと考えた。
タスクの追加、削除、完了に機能を絞ったTodoアプリはちょうどよい難易度だと考え、開発に着手した。

# 使用技術
- React 17.0.2
- TypeScript 4.1.5
- Firebase
  - Firestore
  - Firebase Authorization
- ESLint
- Prettier

## 使用ライブラリ
- Day.js
- Material Ui
- React Router DOM

# 機能一覧
- ユーザー登録、ログイン機能
- タスク管理機能 
  - タスクの追加
  - タスクの削除
  - タスクの完了
