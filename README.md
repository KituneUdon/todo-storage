# 概要
タスクの追加・削除・完了ができる最低限の機能を搭載したTodoアプリ。

# 作成目的
- Reactを用いたアプリ作成
  - 初めてのアプリ作成のため難易度は低いもの

# 使用技術
- React 17.0.2
- TypeScript 4.1.5
- Firebase
  - Firestore
  - Firebase Authorization
- ESLint
- Prettier
- Day.js
- Material Ui
- React Router DOM

# 機能一覧
- ユーザー登録、ログイン機能
- タスク管理機能 
  - タスクの追加
  - タスクの削除
  - タスクの完了
  - タスクの変更
